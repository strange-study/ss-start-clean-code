## 2. 스터디 유형

## 3. 진행 방식
- 1주에 2챕터 씩, 총 8주 과정으로 진행 예정
- 주간 메인 발표자는 해당 주의 진도를 정리 및 발표 (적어도 10분?)
  - 형식은 자유 (ppt, .md, notion, txt ...)
- 주간 과제: 공부 로그 남겨서 공유
  - 공부 로그는 이슈에 코멘트로 남기기
  - 형식은 자유 (ppt, .md, notion, txt ...)
- 결석 1회 / 지각 1회 허용
- 미리 참여자는 돈을 낸 후, 지각/결석/과제미제출 에 따른 금액 차감 (협의 필요)
  - 남은 금액은 마지막 환급
  - 차감 금액은 이후 마지막 날에 맛있는 거 먹을 때 사용 🎉
- 1회독이 목적이므로 실습은 선택 사항
